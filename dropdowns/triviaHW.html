Fun Facts About Resonite!


Resonite started as a game known as NeosVR which was much more limited than Resonite is and was co owned. This game was highly praised for how revolutionary it was but because of one of the developers wanting to add crypto currency to the game the two owners split paths, in turn Resonite was born!

Yellow Dog Man Studios!

This studio is run by Frooxius who was the main developer and co owner of NeosVR as the engine it runs in is coined "Froox Engine" which is highly mutlithreaded but restricted by unity until the awwdio system gets finished.



YES, You very much can code in resonite.

Resonite brands itself with a in house built coding language called ProtoFlux which is node based and running in a virtual machine along side the game!
